# microfront-seed-app

## ¿Cómo iniciar?
Este repositorio consta de 2 proyectos llamados host-app y root:
- host-app
Emula la aplicación en la cual se desea instalar el microfront.
- root
Microfront.

### Configuración
Para comenzar la configuración del proyecto necesitas 2 consolas.

#### Consola 1: 
- yarn install 
- yarn start 

#### Consola 2:
- cd host-app
- yarn install
- yarn start
